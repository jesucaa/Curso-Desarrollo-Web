# 🌐 Curso de Desarrollo Web Completo – Mi Inicio en el Mundo del Desarrollo Web

Este repositorio contiene mi progreso y proyectos realizados durante el **curso técnico y completo de desarrollo web**, impartido por **Juan Pablo De la Torre Valdez** de su website [Código con Juan](https://codigoconjuan.com). Aquí documento mi aprendizaje desde las bases hasta la creación de aplicaciones dinámicas.

## 🚀 Tecnologías y Herramientas

He estructurado mi aprendizaje en torno a estas tecnologías fundamentales:

* **Lenguajes de Programación:** JavaScript (ES6+) y PHP (POO & MVC).
* **Maquetación y Estilos:** HTML5, CSS3 y SASS (Metodologías BEM y Módulos).
* **Base de Datos:** MySQL (Lenguaje SQL y Modelado Relacional).
* **Intercambio de Datos:** Fetch API (Consumo de servicios y comunicación asíncrona).

## 💻 Proyectos Finales

| Proyecto | Descripción | Stack Principal |
| :--- | :--- | :--- |
| **DevWebCamp** | Plataforma de conferencias con registro y pagos. | PHP, MySQL, SASS |
| **AppSalon** | Gestión de citas para salón de belleza. | JavaScript, PHP, MySQL |
| **UpTask** | Administrador de proyectos dinámico. | Fetch API, PHP, MySQL |
| **Bienes Raíces** | Portal inmobiliario con área de administración. | PHP, MySQL, CSS |

## 🧠 Capacidades Desarrolladas

* **Arquitectura:** Implementación del patrón **Model-View-Controller (MVC)** y creación de Frameworks base.
* **Seguridad:** Autenticación de usuarios, manejo de sesiones y **Hash de Passwords**.
* **E-commerce:** Integración de pagos con la API de PayPal.
* **Backend:** Gestión de bases de datos relacionales y envío de emails.
