# 🌐 Curso de Desarrollo Web Completo – Mi Inicio en el Mundo del Desarrollo Web

> 📚 Este repositorio contiene todos los proyectos y prácticas que realicé durante mi **primer curso técnico y completo de desarrollo web**, impartido por **Juan Pablo De la Torre Valdez** de su website [Código con Juan](https://codigoconjuan.com).

---

## 🚀 Acerca del Curso

Este curso representa mi **inicio en el mundo del desarrollo web**. Es un programa intensivo y práctico que abarca **más de 10 lenguajes y tecnologías**, y que culmina con varios **proyectos completos**, aplicando lo aprendido de principio a fin.

El enfoque del curso es **aprender creando proyectos reales**, desarrollando sitios y aplicaciones web dinámicas, seguras y modernas.

---

## 🧠 Tecnologías y Lenguajes Aprendidos

Durante el curso trabajé con las siguientes tecnologías:

- **HTML5** – Estructura semántica de sitios web.  
- **CSS3 / SASS** – Estilos modernos, responsive design y preprocesadores.  
- **JavaScript (ES6)** – Interactividad y manipulación dinámica del DOM.  
- **Fetch API (antes AJAX)** – Conexión entre frontend y backend.  
- **PHP** – Programación backend, programación orientada a objetos (POO) y patrón MVC.  
- **MySQL / SQL** – Creación, modelado y consultas en bases de datos.  
- **Workflows** – Automatización y buenas prácticas de desarrollo.  
- **API’s** – Integración de servicios externos y comunicación entre aplicaciones.

---

## 💻 Proyectos Desarrollados

El curso incluye **4 proyectos finales completos**, además de varios ejercicios prácticos.  
Cada proyecto integra múltiples tecnologías aprendidas:

| Proyecto | Descripción | Tecnologías |
|-----------|--------------|-------------|
| 🏕 **DevWebCamp** | Sitio web para una conferencia de diseño con registro y pagos en línea. | HTML, CSS, PHP, MySQL, PayPal API |
| 💇‍♀️ **AppSalon** | Sistema de gestión para un salón de belleza. | PHP, MySQL, JavaScript |
| 📋 **UpTask** | Administrador de proyectos en tiempo real con AJAX y PHP. | JavaScript, Fetch API, PHP, MySQL |
| 🏠 **Bienes Raíces** | Portal inmobiliario con panel de administración dinámico. | PHP, MySQL, HTML, CSS |

---

## 🔐 Temas Avanzados Aprendidos

- Autenticación y registro de usuarios con **hash de contraseñas**.  
- **Recuperación de acceso** y validación de cuentas.  
- Creación de un **framework base** reutilizable en múltiples proyectos.  
- **Área de administración segura** (login, sesiones, roles).  
- Integración de **pagos en línea con PayPal**.  
- **Modelado de bases de datos** relacionales desde cero.  
- **MVC (Model View Controller)** aplicado a proyectos PHP.  
- Envío de **emails automáticos** y manejo de formularios.  
- **Seguridad web** y buenas prácticas de desarrollo.

---

## 🧩 Qué Aprendí Personalmente

- Cómo unir frontend y backend en proyectos reales.  
- La importancia de la organización del código (MVC y modulación).  
- Buenas prácticas de seguridad y validación de datos.  
- A trabajar con servidores locales y conectar PHP con MySQL.  
- A crear **sitios web completamente funcionales**, desde cero.  

Este curso marcó el **inicio de mi camino profesional como desarrollador web**, y consolidó las bases para seguir aprendiendo tecnologías modernas como React, Node.js o Laravel.
